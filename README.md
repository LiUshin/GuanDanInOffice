# 局域网掼蛋游戏 (Lan GuanDan)

这是一个基于 Node.js + React 的局域网掼蛋游戏，支持单机运行并供局域网内其他设备连接。

## 功能特性
- **局域网联机**: 一人开启服务，其他人通过 IP 连接。
- **自动补位**: 人数不足时，点击"开始游戏"会自动添加 AI 机器人。
- **游戏规则**: 经典掼蛋规则 (进贡/还贡暂未包含在 MVP 版本中)，支持 2, 3, 4, 5...A 升级。
- **单文件运行**: 可打包为唯一的 `.exe` 文件，无需安装环境。

## 开发环境运行

1. 安装依赖:
   ```bash
   npm install
   ```

2. 启动开发服务器 (前后端热更新):
   ```bash
   npm run dev
   ```

## 打包发布

1. 构建并打包 Windows 可执行文件:
   ```bash
   npm run package
   ```
   打包完成后，会在 `release/` 目录下生成 `guandan-game.exe`。

## 如何玩

1. **主机 (房主)**:
   - 双击运行 `guandan-game.exe`。
   - 此时会自动启动服务器（默认端口 3000）。
   - 在本机浏览器访问 `http://localhost:3000`。
   - 输入昵称，加入默认房间。
   - 等待其他人加入，或者点击 "开始游戏 (Host)" 强制开始（自动添加 Bot）。

2. **其他玩家**:
   - 确保与主机在同一局域网。
   - 获取主机的局域网 IP 地址 (如 `192.168.1.5`)。
   - 在浏览器访问 `http://192.168.1.5:3000`。
   - 输入昵称加入。

## 项目结构
- `src/server`: Node.js 后端 (Socket.IO, 游戏逻辑)
- `src/client`: React 前端 (UI, 交互)
- `src/shared`: 前后端共用的类型和规则
